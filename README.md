### ✨ About me

Cloud & DevOps Engineer actively transitioning into **MLOps and QDevOps**, with hands-on experience building and deploying cloud-native applications on **AWS and Azure**.

Skilled in Terraform, Docker, Kubernetes, CI/CD pipelines, and cloud security. Recently built a production-ready MLOps pipeline on AWS — from model training with MLflow experiment tracking, to containerized FastAPI deployment on EC2, with full CI/CD automation via GitHub Actions.

Background in **Molecular Engineering** with experience in **Quantum Computing** (Qiskit, Q#, Azure Quantum), bringing strong analytical and problem-solving skills to cloud security and infrastructure challenges.

- 🏅 AWS Solutions Architect Associate
- 🏅 AWS Solutions Developer Associate in progress via TÜV Rheinland
- 🏅 Google Cybersecurity Certificate
- 📚 AZ-104 
- 📚 AZ-500 in progress via TÜV Rheinland
